#  Bank Application (Frontend Project)

##  Description

This is a simple **Bank Application frontend project** developed using HTML, CSS, and JavaScript.
It simulates basic banking operations like user login, registration, and account interaction.

##  Features

*  User Registration
*  Login System
*  User Dashboard
*  Clean UI Design
*  Interactive using JavaScript

##  Technologies Used
* HTML5
* CSS3
* JavaScript

##  Project Structure
* login.html → Login page
* register.html → Registration page
* user.html → User dashboard
* style.css → Styling
* script.js → Functionality

##  How to Run

1. Download or clone the repository
2. Open `login.html` in any browser
3. Start using the application

* Backend integration
